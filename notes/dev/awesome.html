dotfiles config --local status.showUntrackedFiles no

dotfiles add .bashrc
dotfiles commit -m 'Add .bashrc'
dotfiles push

  • 订阅精选的 Newsletter，你可以节省在互联网上搜索和筛选信息的时间。
  • 提供的案例研究、最佳实践和教程等内容，是持续学习和技能提升的宝贵资源。
