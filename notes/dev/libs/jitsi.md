# Jitsi

- [jitsi/libjitsi](https://github.com/jitsi/libjitsi)
  - Advanced Java media library for secure real-time audio/video communication.
  - capture, playback, stream, encode/decode and encrypt audio and video flows
  - audio mixing, handling multiple streams, participation in audio and video conferences.
