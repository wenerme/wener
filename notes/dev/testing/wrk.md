# wrk

```
  -c, --connections <N>  打开的连接数
  -d, --duration    <T>  测试时间
  -t, --threads     <N>  线程数

  -s, --script      <S>  加载 Lua 脚本
  -H, --header      <H>  测试时添加 Header
      --latency          打印延迟统计
      --timeout     <T>  Socket/请求 超时时间
  -v, --version          Print version details

数字参数可使用 SI 单位 (1k, 1M, 1G)
时间参数可使用时间单位 (2s, 2m, 2h)
```
