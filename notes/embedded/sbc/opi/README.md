# Orange Pi (香橙派) Single Board Computers

Orange Pi is a series of open-source single-board computers by Shenzhen Xunlong Software.

- [Orange Pi Official (English)](http://www.orangepi.org/)
- [Orange Pi Official (Chinese)](http://www.orangepi.cn/)

## Board Specifications

### Orange Pi Zero

- **CPU**: Allwinner H2+ Quad-core Cortex-A7 @ 1.2GHz.
- **RAM**: 256MB/512MB DDR3.
- **Connectivity**: 100M Ethernet (PoE support), USB 2.0, UART.
- **Storage**: 2MB SPI Flash + MicroSD slot.
- **Wireless**: XR819 WiFi (802.11 b/g/n).
- **Expansion**: 26-pin Header + 13-pin Function Header (Audio, TV-out, Mic, IR, 2x USB).

### Orange Pi Zero Plus

- **CPU**: Allwinner H5 Quad-core 64-bit Cortex-A53.
- **GPU**: Mali-450.
- **Connectivity**: 1000M (Gigabit) Ethernet.
- **Wireless**: Realtek RTL8189FTV WiFi.
- **Tech**: Higher performance than the standard Zero, suitable for networking tasks.
