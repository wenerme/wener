    • 发布信息包 - dissemination information package

档案归档内容​

  • 收集 (Collection)
    • 收集需要归档的文件和资料，包括纸质文件和电子文件。
    • 将纸质文件进行数字化扫描，转化为电子文件。
  • 整理 (Organization)
    • 对收集到的文件进行分类、编号、排序和编目，确保文件有序、系统地存放。
    • 对电子文件进行分类、编号、排序和编目，存放在电子档案系统中。
  • 著录 (Cataloging)
    • 对文件的基本信息进行记录和标注，包括标题、作者、日期、内容摘要等。
    • 在电子档案系统中记录和标注文件的基本信息。
  • 编目 (Indexing)
    • 系统地组织和分类文献或档案，使其易于检索和管理。包括分类、编排和索引等工作。
    • 在电子档案系统中进行编目。
  • 保存 (Storage)
    • 将整理好的文件存放在适当的档案库房或电子档案系统中，确保文件的安全和长期保存。
    • 将电子文件存放在电子档案系统中。
  • 检索 (Retrieval)
    • 建立检索系统，方便用户快速查找和利用档案信息。
    • 使用电子档案系统的检索功能。
  • 维护 (Maintenance)
    • 定期检查和维护档案，防止文件损坏或丢失，确保档案的完整性和可用性。
    • 定期检查和维护电子档案系统。
  • 销毁 (Destruction)
    • 对于超过保存期限且无保存价值的档案，按照规定程序进行销毁处理。
    • 对于超过保存期限且无保存价值的电子档案，按照规定程序在电子档案系统中进行销毁处理。

物品​

物品 ≧ 实物 ≧ 实体档案 ≧ 实物档案
  • 物品
  • 实物
    • 区别于电子类的物品
    • 未归档的实物，归档前叫实物，归档后叫实物档案。
  • 实体档案
    • 区别于电子档案的档案分类、有纸质类、实物类、声像类
  • 实物档案
    • 实体档案 的一种
    • 是指按照实物档案编制档案分类号并进行管理的一类档案；
    • 未按照实物档案编制档号管理的其他来源的总称实物，实物档啊和实物可以同时共置于同一个库房管理，实物库房保管的不一定都是实物档案。

页码/页数/页号/起止页号​

  • 页码 - Page Number
    • 印刷物本身印刷上去的页码，一般是左右都有，一般不管页面上有没有文字
  • 页数 - Page Count
    • 纸质的张数，正反两面一张纸，只算页数一张
  • 页号 - Page Serial Number/Page Sequence Number
    • 整理和加工以及著录档案材料是，人为手工标注上去的每页材料号码流水，一般是无字的页面不编号。
  • 起止页号 - Start Page Number - End Page Number
    • 指以卷归档的档案，其卷内文件目录里标注每件文件的开始页号-结束页号。

以件归档的材料页号编制规则：纸质页号为1到n，起止页号为1-n。 以卷归档的材料页号编制规则为1到n，而在卷内文件目录里体现的是起止页号：每件文件的开始页号-结束页号。

序号/文号/案卷号/件号/文件号/流水号/页号/盒号/柜号/列号/全宗号​

档案管理中的编号系统

  • 序号（Sequence Number）
    • 定义：表示文件或档案的排列顺序。
    • 特点：仅表示顺序，无其他意义。
  • 文号（Document Number）
    • 定义：公文编号，用于分类和检索。
    • 特点：唯一标识文件，包含发文机关、年份、顺序号。
  • 案卷号（Volume Number）
    • 定义：立卷归档后编制的流水号。
    • 特点：仅适用于卷归档档案。
  • 件号/文件号（Item Number/Document Number）
    • 定义：每件档案的唯一编号。
    • 特点：文件级别编号，以卷归档的文件有件号。
  • 流水号（Serial Number）
    • 定义：表示档案或文件的排列顺序号。
    • 特点：与序号类似，按时间或提交顺序生成。
  • 盒号（Box Number）
    • 定义：档案盒的编号，标识其物理位置。
    • 特点：用于查找档案盒在存储系统中的位置。
  • 页号（Page Number）
    • 定义：材料归档前标注在右上角的页数编号。
    • 特点：自然数顺序，确保页数清晰。
  • 柜号（Cabinet Number）
    • 定义：档案柜的编号。
    • 特点：标识档案柜的位置。
  • 列号（Row Number）
    • 定义：档案密集架中列的编号。
    • 特点：用于进一步细化档案存储位置。
  • 全宗号（Fonds Number）
    • 定义：单位或机构形成档案的分类编号。
    • 特点：标识档案所属的单位或时期。

编写/编著/编撰/编纂/编修/编辑/著录/标引​

  • 编写（Writing）
    • 定义：富有原创性的创作过程。
    • 特点：侧重原创内容的撰写。
  • 编著（Compilation and Writing）
    • 定义：通常指专业类或专著类的撰写过程。
    • 特点：专业领域的系统性创作。
  • 编撰（Compilation with Fictional Content）
    • 定义：包含虚构或未经考证内容的编写过程。
    • 特点：虚构性较强，适用于文学或非学术作品。
  • 编纂（Compilation and Organization）
    • 定义：系统性收集、整理已有材料的过程，常用于志书、年鉴等工具书。
    • 特点：重在整理已有资料。
  • 编修（Revision and Compilation）
    • 定义：修订内容较多的编辑活动。
    • 特点：侧重对原有内容的大规模修订。
  • 编辑（Editing）
    • 定义：职业性工作，按标准使内容规范统一。
    • 特点：规范化处理内容，使其符合出版或发布标准。
  • 著录（Cataloging）
    • 定义：概括或标注档案案卷和文件信息的工作，便于查询。
    • 特点：针对档案信息的记录与概括。
  • 标引（Indexing）
    • 定义：对著录信息的进一步详细标注，便于精确查询。
    • 特点：细化档案信息，提升检索效率。
  • 编目（Cataloging/Classification）
    • 定义：系统地组织和分类文献或档案的过程，使其易于检索和管理。
    • 特点：不仅包括著录，还涉及分类、编排和索引等工作。
    • 例如：图书馆编目、档案编目。

文稿/文本/正本/副本/修订本/试行本​

  • 文稿（Manuscript/Draft）
    • 定义：文稿是文件正式形成前的历次稿件，包含草稿和定稿两个阶段。
      • 草稿：初步撰写的文件版本，通常未经正式审查或批准。
      • 定稿：最终确定的稿件，经过审定后作为正式文本。
        • Final Draft
        • Finalized Version
  • 文本（Text/Version）
    • 定义：文本是指一个文件的正式版本，可能依据不同用途分为多个不同版本，如正本、副本、试行本、修订本等。
    • 特点：文本是经过确定后的文件内容的载体，可根据不同使用场景有不同的形式。
  • 正本（Original Copy）
    • 定义：正本是文件的正式且原始的版本，具有法律效力或最终权威性。
    • 特点：通常用于正式场合，如发布、签署、备案等。
  • 副本（Duplicate/Copy）
    • 定义：副本是正本的复印件或复制版本，用于参考或分发，但不具有正本的法律效力。
    • 特点：副本用于传阅、备查或工作中的参考，不用于正式用途。
  • 修订本（Revised Edition）
    • 定义：修订本是对原文件内容进行修改或补充后的版本。
    • 特点：在原有文本的基础上进行修订，以反映最新的内容或更正错误。
  • 试行本（Trial Version）
    • 定义：试行本是文件的试行版本，通常用于在特定时间或范围内试行，待验证后再正式发布。
    • 特点：试行本在验证后可能成为正式版本，也可能在试行期结束后进行修改。

关键词/主题词/检索词​

  • 关键词（Keywords）
    • 定义：用于模糊查询的词汇，通常反映文件或档案的核心内容。
    • 特点：关键词适用于广泛的检索，帮助用户快速找到相关档案。
  • 主题词（Subject Terms）
    • 定义：公文中常用的词汇，用于帮助档案的分类和检索。
    • 特点：主题词通常是公文内容的核心概念，随着检索工具的进步，现代公文已较少标注主题词。
  • 检索词（Search Terms）
    • 定义：用于档案查询和检索的统称，包含关键词和主题词等。
    • 特点：检索词用于提升查询效率，帮助快速定位相关档案内容。

专业档案/专门档案/专题档案/专项档案​

基于档案的用途、内容、来源以及形成背景来进行的划分

  • 专业档案（Professional Archives）
    • 定义：属于专门档案的一类，反映某一专门领域的档案，突出材料的专业性。
    • 特点：涵盖特定专业领域，如玉器档案、病例档案、植物档案、教学档案等，主要用于记录某一领域的专业信息。
  • 专门档案（Specialized Archives）
    • 定义：泛指特定领域或专业领域形成的档案，反映该领域的专门信息。
    • 特点：包括广泛的专业领域档案，如电力档案、基建档案等，是某一领域活动或事务的专门记录。
  • 专题档案（Thematic Archives）
    • 定义：某一活动或事件全过程形成的材料，通常是专题档案材料的简称。
    • 特点：并非严格意义上的档案，材料可能包含人为制作或复制的内容，真实性不一定完全可靠。专题档案有参考价值，目录通常单独制作，不与档案馆的正式档案目录混合。
  • 专项档案（Special Project Archives）
    • 定义：指针对某一专项任务、项目或计划所形成的档案，记录该专项工作的全部过程。
    • 特点：通常围绕特定项目或专项工作，如专项审计档案、专项工程档案，全面记录项目的背景、实施过程和结果。

机构/问题代码​

  • 机构 - Provenance/Creator
    • 办公室、人事部、财务处等
    • 来源单位，便于按单位或部门管理和归档
  • 问题/主题 - Subject/Function
    • 党务、财务、行政等问题类别
    • 内容主题，便于从主题维度检索
    • “问题代码”可能直接反映了档案的核心内容或“案由”（case reason/subject matter）
  • [机构代码A] -> [该机构下的问题代码X] -> [具体文件]
  • 办公室（机构） -> 党务工作（问题）
  • 财务处（机构） -> 年度预算（问题）

Archival vs Archive​

  • Archival - 档案的
    • 形容词
    • 与档案相关的，或用于描述档案管理的过程和方法
    • 例如：archival quality (档案级质量), archival storage (档案存储)
  • Archive - 档案、归档
    • 名词或动词
    • 名词：指档案或档案库，存放文件和记录的地方
      • 例如：The archive contains historical documents. (档案馆包含历史文件)
    • 动词：指将文件或记录归档的动作
      • 例如：We need to archive these documents. (我们需要将这些文件归档)
