WireGuard Awesome

tip
  • 值得关注 Wiretrustee
  • 目前只有 Tailscale 的 移动端支持 Mesh
  • Vanilla WireGuard
    • wireguard-go
      • MIT, Golang
      • Go 实现 wg
    • wireguard-apple
      • MIT, Golang+Swift
    • wireguard-android
    • wireguard-windows
      • MIT, Golang, C
      • Windows 实现，基于 wireguard-nt
    • wireguard-nt
      • GPLv2, C
      • Windows 内核实现
      • NT Kernel
      • 预构建 binary License 更松
      • https://download.wireguard.com/wireguard-nt/
    • wireguard-linux
      • Linux kernel
    • wgctrl-go
    • wintun
      • Windows 下 tun 设备
  • cloudflare/boringtun
    • BSD-3, Rust
    • Userspace WireGuard® Implementation in Rust
  • tonarino/innernet
    • MIT, Rust
  • Place1/wg-access-server
    • MIT, Go
    • WireGuard VPN solution + Web UI
    • HN
  • subspacecommunity/subspace
    • WireGuard VPN server GUI
  • firezone/firezone
    • Apache-2.0, Elixir+Ruby
    • WebUI
    • nftables
    • 兼容官方 WireGuard
  • weejewel/wg-easy
    • JS
    • WireGuard VPN + Web-based Admin UI
  • aramperes/onetun
    • MIT, Rust
    • Access your WireGuard network from anywhere
  • freifunkMUC/wg-access-server
    • MIT, Go
    • WebUI
  • database64128/swgp-go
    • Simple WireGuard proxy with minimal overhead for WireGuard traffic

WireGuard Mesh​

  • wiretrustee/wiretrustee Wireguard+Mesh
    • BSD-3, Go
    • WebRTC, STUN, TURN
  • gravitl/netmaker
    • SSPL, Golang
    • Mesh
      • 没有也不考虑支持移动端
    • Admin UI, Private DNS, OAuth, ACL
  • tailscale/tailscale WireGuard+2FA
    • BSD-3
    • 依赖 官方 coordinate 服务
  • juanfont/headscale
    • self-hosted Tailscale controller server
    • tailscale 目前移动端不支持修改 controller
  • HarvsG/WireGuardMeshes Compare WireGuard Mesh Tools
