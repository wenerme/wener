# Google Drive Utils

- [gd-utils/gd-utils](https://github.com/iwestlin/gd-utils)
  - [Compare with AutoRclone and gclone](https://github.com/iwestlin/gd-utils/blob/master/compare.md)
  - Not just the fastest Google Drive transfer tool.
  - Features:
    - Count files in any directory, export as html/table/json. Supports resume.
    - Copy files between directories, filter by size, supports resume.
    - Deduplication (md5), delete empty directories.
    - Deploy as HTTP API.
    - Telegram bot support.
