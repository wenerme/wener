# Vtiger CRM

- [SourceForge: Vtiger CRM](https://sourceforge.net/projects/vtigercrm/)

## Features

- End to end sales cycle management (Campaigns, Leads, Potentials, Quotes, Invoices).
- Automation support (Customer Portal, Support Tickets).
- Data import/export (CSV, Web-to-Lead forms).
- Reports & User Dashboards.
- Role based access control.
- Mobile applications.
- Workflows, Tasks, Project Management.
- Plugins: Outlook, Thunderbird, Firefox, Gmail.
- Extensions Marketplace.
