# 会计信息系统 (AIS)

- [Accounting Information System (Wikipedia)](https://en.wikipedia.org/wiki/Accounting_information_system) - 会计信息系统
- [Accounting Software (Wikipedia)](https://en.wikipedia.org/wiki/Accounting_software) - 会计软件

## 核心模块 (Core modules)

- **Accounts receivable (应收账款)**: 记录公司收到的款项。
- **Accounts payable (应付账款)**: 记录公司账单及支付欠款。
- **General ledger (总账)**: 公司的总账簿。
- **Billing (开票)**: 向客户生成发票。
- **Stock/inventory (库存)**: 控制库存。
- **Purchase order (采购订单)**: 订购库存。
- **Sales order (销售订单)**: 记录客户的库存供应订单。
- **Bookkeeping (记账)**: 记录收款和付款。
- **Financial close management (财务结账管理)**: 会计团队在指定期间结束时核实和调整账户余额。

## 非核心模块 (Non-core modules)

- **Debt collection (债务催收)**: 追踪逾期账单的催收工作（有时作为应收账款的一部分）。
- **Electronic payment processing (电子支付处理)**
- **Expense (报销/费用)**: 录入员工的业务相关费用。
- **Inquiries (查询)**: 在屏幕上查找信息，不进行编辑或添加。
- **Payroll (薪资)**: 追踪工资、薪金及相关税费。
- **Reports (报表)**: 打印数据。
- **Timesheet (工时表)**: 专业人员（如律师和顾问）记录工作时间以便向客户收费。
- **Purchase requisition (采购申请)**: 发起、批准和追踪采购订单请求。
- **Reconciliation (对账)**: 比较交易双方的记录以确保持致性。
- Drill down (向下钻取)
- Journals (日记账)
- Departmental accounting (部门会计)
- Support for value added taxation (增值税支持)
- Calculation of statutory holdback (法定保留金计算)
- Late payment reminders (逾期付款提醒)
- Bank feed integration (银行流水集成)
- Document attachment system (文档附件系统)
- Document/Journal approval system (文档/日记账审批系统)
