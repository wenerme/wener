InfiniBand

caution
  • mtu > 1500 可能会有问题
  • 2019 NVIDIA to Acquire Mellanox for $6.9 Billion
  • InfiniBand
  • Current message level
    • https://www.kernel.org/doc/Documentation/networking/netif-msg.txt
  • Arch
    • https://wiki.archlinux.org/index.php/InfiniBand
    • https://wiki.archlinux.org/index.php/Jumbo_frames
  • 10 Gig Eth
    • 不需要 IB 交换机
    • Mellanox 带 _en 的即为 ethernet 设备
  • jumbo frames
    • MTU 大于 1500 的 Ethernet 帧
    • 最多 9000, 根据设备和网络结构可能有所不同
    • 可能增加延时, 但是能大量提升网络传输速度
  • mellanox
    • MLNX_EN_Linux_README.txt
    • linux mlx4_en.h
  • 参考
    • infinibandta
    • How to fine tune TCP performance on Linux with a 10Gb fiber connection
    • How to achieve low latency with 10Gbps Ethernet
  • 注意
    • 在 /etc/network/interface 下配置的顺序会影响默认网卡, 将 ib 配置在第一位即可
  • SM - subnet manager

  • 速度没上去
    • 确定是走的正确的网卡, 一般 ib 不是 eth0
# 确保默认路由是 ib 卡
ip ro get 8.8.8.8

# 如果不是, 则替换默认路由
# 假设 eth2 为 ib 卡
ip route replace default via 192.168.1.1 dev eth2
