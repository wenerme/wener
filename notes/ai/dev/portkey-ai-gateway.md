# Portkey-AI/gateway

- [Portkey-AI/gateway](https://github.com/Portkey-AI/gateway)

  - 每次请求带上配置
- 不依赖 DB
- 可使用 Redis 配置

- 开源版只有日志查看 UI

```bash
# run by npm
# http://localhost:8787/public/
npx @portkey-ai/gateway
```
