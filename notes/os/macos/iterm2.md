| 快捷键                    | 功能              |
| ------------------------- | ----------------- |
| ⌘ + T                     | 新建标签页        |
| ⌘ + N                     | 新建窗口          |
| ⌘ + D                     | 垂直分屏          |
| ⌘ + Shift + D             | 水平分屏          |
| ⌘ + ] / [                 | 切换分屏          |
| ⌘ + Option + 方向键       | 切换分屏          |
| ⌘ + W                     | 关闭当前标签/分屏 |
| ⌘ + 数字                  | 切换到第 n 个标签 |
| ⌘ + 左/右                 | 切换标签          |
| ⌘ + Enter                 | 全屏切换          |
| ⌘ + Shift + Enter         | 最大化当前分屏    |
| ⌘ + K                     | 清屏              |
| ⌘ + F                     | 查找              |
| ⌘ + Shift + H             | 粘贴历史          |
| ⌘ + ;                     | 自动补全          |
| ⌘ + Shift + E             | 即时回放          |
| ⌘ + Shift + Option + 拖动 | 块选择            |
| ⌘ + Option + B            | 即时回放          |
| ⌘ + Shift + O             | 打开最近目录      |
| ⌘ + /                     | 高亮光标位置      |
