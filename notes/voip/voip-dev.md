# VoIP 开发

## JavaScript/WebRTC

- [onsip/sip.js](https://github.com/onsip/sip.js)
  - MIT, TypeScript
  - WebRTC SIP 客户端库
  - 支持：注册、音视频通话、IM、Presence、屏幕共享、DTMF
  - [sipjs.com](https://sipjs.com/)
  - [Asterisk 配置指南](https://sipjs.com/guides/server-configuration/asterisk/)

- [davehorton/drachtio-srf](https://github.com/davehorton/drachtio-srf)
  - MIT, JavaScript
  - Node.js SIP Server 框架
  - Drachtio Signaling Resource Framework

## Go

- [pion/webrtc](https://github.com/pion/webrtc)
  - MIT, Go
  - 纯 Go 实现的 WebRTC
- [emiago/sipgo](https://github.com/emiago/sipgo)
  - BSD-2, Go
  - SIP 协议库

## CPaaS 平台

- [OnSIP](https://developer.onsip.com/)
  - SIP.js 创建者
  - 提供 CPaaS API
