# Wicket

- [apache/wicket](https://github.com/apache/wicket)
  - Apache-2.0, Java, Web Framework
  - Component-based web application framework for Java.
- 服务端前端组件化开发
  - 类似于 Vaddin、GWT
  - 类似于 React 的 SSR

## Notes

- Page
  - 有版本概念
    - 路径上 `/?n`
  - 有状态概念 - 是否依赖 Session
    - 需要进行序列化
