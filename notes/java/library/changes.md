# CHANGELOG

## Guava

### 22

- 2017.5.22
- 专注于 Android 方便的提升

### 21

- 2017.1.12
- 版本要求 **Java 8**
- 大多数类型都和 Java 8 类型兼容了
- 添加了 Stream 工具类

### 20

- 2016.10.28
- [Release20](https://github.com/google/guava/wiki/Release20)
- 添加了 `common.graph` 包
- common.hash
  - 添加了 FarmHash Fingerprint64 和一些 HMAC 算法
- common.math
  - 主要添加了统计相关的类型
  - Quantiles
  - Stats 和 StatsAccumulator, PairedStats 和 PairedStatsAccumulator
  - LinearTransformation
