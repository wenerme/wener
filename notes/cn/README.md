## 基础概念

- CTID - 居民身份证网上凭证
  - 公安部第一研究所可信身份认证体系相关能力
  - 常见于政务、金融、实名核验场景
- 五险一金
  - 养老保险
  - 医疗保险
  - 失业保险
  - 工伤保险
  - 生育保险
  - 住房公积金
- 三险一金
  - 基本养老保险
  - 基本医疗保险
  - 失业保险
  - 住房公积金
