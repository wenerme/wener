# SQL

## Frameworks & Libraries

- [Apache Calcite](https://calcite.apache.org/)
  - **The** standard SQL parser, validator and JDBC driver.
  - Query optimization (relational algebra, cost model).
  - Federated query (push down computation).
  - [Paper](https://arxiv.org/pdf/1802.10233.pdf): A Foundational Framework for Optimized Query Processing Over Heterogeneous Data Sources.
- [araddon/qlbridge](https://github.com/araddon/qlbridge)
  - Go SQL expression VM.
- [dataux/dataux](https://github.com/dataux/dataux)
  - Federated MySQL-compatible proxy (Elasticsearch, Mongo, Cassandra, etc.)
