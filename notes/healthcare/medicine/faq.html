Medical FAQ

warning

药物信息只用于理解常见药物类别，不构成诊断或处方建议。用药前应阅读说明书/Drug Facts label；孕期、哺乳期、儿童、老人、肝肾功能异常、慢病、多药合用、过敏史明确者，先问医生或药师。

用药安全原则​

  • 看 active ingredient / 通用名 / 成分名，不要只看商品名；复方感冒药很容易和退烧止痛药重复成分。
  • 商品名有地区差异，同一成分可能有多个品牌；优先核对中文通用名和英文通用名。
  • 不要超说明书剂量、频次、天数；症状持续或加重应就医。
  • 抗生素、激素、降压药、降糖药、抗凝药、精神科药物不要自行开始/停药。
  • 儿童慎用成人药；2 岁以下不要自行用咳嗽/感冒药；儿童/青少年病毒感染时不要用 aspirin。
  • 有高血压、心脏病、青光眼、前列腺增生者，慎用鼻减充血药。
  • NSAIDs 如 ibuprofen / 布洛芬、naproxen / 萘普生可能刺激胃、影响肾功能、增加出血风险；胃溃疡、肾病、抗凝药使用者需谨慎。
  • Acetaminophen / paracetamol / 对乙酰氨基酚/扑热息痛过量可伤肝；感冒药、止痛药、退烧药里可能重复包含。

发热、头痛、轻中度疼痛​

项目内容
常见药物acetaminophen/paracetamol / 对乙酰氨基酚/扑热息痛
常见商品名Tylenol / 泰诺、Panadol / 必理通、泰诺林
主要用途退热、头痛、轻中度疼痛
注意注意每日总量；肝病、长期饮酒、复方感冒药同用时更容易超量
  • FDA 标注成人和 12 岁以上 24 小时总量不应超过 4000 mg；不同商品说明书可能更保守，优先按说明书。
  • 儿童剂量按体重和说明书/医生建议，不要用成人剂量估算。
  • 如果高热持续、精神差、呼吸困难、皮疹、颈部强直，应及时就医。

疼痛伴炎症、牙痛、痛经、关节痛​

项目内容
常见药物ibuprofen / 布洛芬；naproxen / 萘普生
常见商品名Advil、Motrin、芬必得；Aleve
药物类别NSAIDs / 非甾体抗炎药
注意胃病、肾病、抗凝药、孕 20 周后/孕晚期慎用；不要多种 NSAIDs 叠加
  • FDA 建议孕 20 周或之后避免 NSAIDs，除非医生明确建议；30 周后通常应避免。
  • 疼痛伴红肿热痛、外伤变形、牙龈化脓、痛经突然加重，应找原因而不是只止痛。

阿司匹林相关​

项目内容
常见药物aspirin / 阿司匹林
常见商品名Bayer / 拜耳、拜阿司匹灵
常见用途抗血小板、止痛退热，但用途和剂量差异很大
注意儿童/青少年病毒感染时避免；胃出血、抗凝药、手术前需谨慎
  • 心脑血管预防用 aspirin 是否需要长期吃，应由医生根据出血风险和获益判断。

过敏性鼻炎、荨麻疹、皮肤痒​

项目内容
常见药物loratadine / 氯雷他定；cetirizine / 西替利嗪；fexofenadine / 非索非那定
常见商品名Claritin / 开瑞坦、Zyrtec / 仙特明、Allegra
药物类别第二代 antihistamine / 抗组胺药
注意相对少嗜睡，但仍可能困倦；驾驶/饮酒需谨慎
  • 荨麻疹伴喉头紧、喘、嘴唇舌头肿、低血压感，应按过敏急症处理。

夜间过敏、晕车、短期助眠​

项目内容
常见药物diphenhydramine / 苯海拉明；dimenhydrinate / 茶苯海明
常见商品名Benadryl、Dramamine、乘晕宁
药物类别第一代 antihistamine / 抗组胺药
注意嗜睡、口干、尿潴留更明显；老人、青光眼、前列腺增生慎用
  • 不建议把第一代抗组胺药作为长期助眠方案。

鼻塞​

项目内容
常见药物pseudoephedrine / 伪麻黄碱；oxymetazoline / 羟甲唑啉；saline spray / 生理盐水喷雾
常见商品名Sudafed、Afrin、诺斯清/海盐水喷雾
主要用途缓解鼻塞、鼻腔干燥
注意高血压/心脏病慎用；oxymetazoline 连用通常不超过 3 天，避免反跳性鼻塞
  • FDA 2024 提议移除口服 phenylephrine / 去氧肾上腺素作为 OTC 鼻减充血成分，因为证据显示口服形式对鼻塞无效；该提议不影响鼻喷 phenylephrine。
  • 过敏性鼻炎长期反复，应考虑鼻喷激素、抗组胺药、环境控制等系统处理，最好咨询医生。

咳嗽​

项目内容
常见药物dextromethorphan / 右美沙芬；guaifenesin / 愈创甘油醚
常见商品名Robitussin DM、Mucinex、右美沙芬糖浆
主要用途DXM 镇咳；guaifenesin 祛痰
注意持续咳嗽、喘、胸痛、血痰、发热不退、老人儿童症状重，应就医
  • 感冒咳嗽多为自限性；补水、休息、湿化空气也很重要。
  • 不要给 2 岁以下儿童自行用咳嗽/感冒药。

咽喉痛、咽喉发炎​

项目内容
常见药物对乙酰氨基酚/布洛芬；lozenges / 含片；saline gargle / 温盐水漱口
常见商品名西瓜霜含片、金嗓子、Strepsils
主要用途缓解疼痛、刺激感、干燥感
注意多数为病毒或刺激；高热、脓点、呼吸困难、吞咽困难需就医
  • 咽痛不等于细菌感染，不一定需要抗生素。
  • 高热、扁桃体脓点、颈部淋巴结肿痛、皮疹，或症状超过数天不缓解，应就医排除 strep throat 等。

慢性咽炎​

项目内容
常见方向处理诱因：postnasal drip、过敏、胃食管反流、吸烟/粉尘、用嗓过度
常见辅助生理盐水鼻喷、含片、补水、减少刺激、反流管理
不建议长期自行用抗生素或含激素喷剂
警示持续声音嘶哑、吞咽困难、咯血、体重下降、单侧疼痛或颈部包块需尽快就医

胃酸、烧心​

项目内容
常见药物antacid / 抗酸药；alginate / 海藻酸盐；famotidine / 法莫替丁；omeprazole / 奥美拉唑
常见商品名Tums、Gaviscon、Pepcid、Prilosec、达喜、洛赛克
主要用途胃酸、烧心、反流症状短期缓解
注意胸痛需排除心梗；长期反酸、吞咽困难、黑便、体重下降需就医
  • PPI 如 omeprazole / 奥美拉唑不建议无评估长期吃。

腹泻​

项目内容
常见药物ORS / 口服补液盐；loperamide / 洛哌丁胺；bismuth subsalicylate / 次水杨酸铋
常见商品名Imodium / 易蒙停、Pepto-Bismol、口服补液盐 III
主要用途补液、防脱水；短期减少腹泻次数
注意血便、高热、明显腹痛、脱水、儿童/老人慎用止泻药，应就医
  • 腹泻首要是补液和电解质，不是立刻止泻。
  • 抗生素相关腹泻、旅行后腹泻、持续超过 2 天的成人腹泻应提高警惕。

便秘​

项目内容
常见药物fiber/psyllium / 膳食纤维/车前子壳；PEG/macrogol / 聚乙二醇；lactulose / 乳果糖；senna / 番泻苷；bisacodyl / 比沙可啶
常见商品名Metamucil、MiraLAX、Duphalac / 杜密克、Dulcolax
主要用途增加粪便体积、软化粪便或刺激排便
注意先水分/膳食纤维/活动；腹痛呕吐、便血、体重下降、习惯突然改变需就医
  • 不建议长期依赖刺激性泻药。

脚气、足癣、真菌感染​

项目内容
常见药物terbinafine / 特比萘芬；clotrimazole / 克霉唑；miconazole / 咪康唑；tolnaftate / 托萘酯
常见商品名Lamisil / 兰美抒、Lotrimin、Daktarin / 达克宁
主要用途足癣、体癣等浅表真菌感染
注意保持干燥；症状消失后继续一段时间；2–4 周无效就医
  • 关键不是只抹药：脚趾缝保持干燥、勤换袜、鞋子通风。
  • 如果反复、趾甲变厚变色、糖尿病/免疫低下，应就医。

湿疹、皮炎轻度瘙痒​

项目内容
常见药物emollient / 润肤剂；hydrocortisone / 氢化可的松；antihistamine / 抗组胺药
常见商品名Vaseline / 凡士林、Cortaid、氢化可的松乳膏；尤卓尔为丁酸氢化可的松，按说明书/医嘱短期使用
主要用途保湿屏障修复、短期缓解轻度炎症和瘙痒
注意面部/眼周/儿童/感染破溃慎用激素；大片扩散或渗液需就医
  • 保湿是基础：减少热水、香精、刺激性清洁剂。
  • 外用激素按部位、强度、天数使用，不建议长期自行反复用。

肌肉关节局部疼痛​

项目内容
常见药物diclofenac gel / 双氯芬酸凝胶；menthol/camphor / 薄荷脑/樟脑外用
常见商品名Voltaren Gel / 扶他林、曼秀雷敦/薄荷膏类
主要用途局部肌肉、软组织、关节疼痛
注意外用 NSAID 仍可能有禁忌；破损皮肤不用

伤口清洁、消毒​

项目内容
常见药物saline / 生理盐水；povidone-iodine / 聚维酮碘/碘伏；chlorhexidine / 氯己定/洗必泰
常见商品名Betadine、Hibiscrub、碘伏棉签
主要用途冲洗、清洁、皮肤消毒
注意深伤口、动物咬伤、污染伤、红肿热痛化脓就医
  • 普通浅表伤口优先清洁和覆盖；不要把刺激性消毒剂反复灌入深部组织。

眼干​

项目内容
常见药物artificial tears / 人工泪液
常见商品名Refresh、Systane / 思然、海露
主要用途干眼、异物感、屏幕用眼后不适
注意眼痛、畏光、视力下降、外伤不要自行处理

口腔溃疡​

项目内容
常见药物lidocaine gel / 利多卡因凝胶；triamcinolone acetonide / 曲安奈德口腔软膏；局部贴片；漱口水
常见商品名西瓜霜、意可贴、Kenalog in Orabase
主要用途局部止痛、保护创面、减少刺激；含糖皮质激素制剂用于减轻局部炎症
注意反复/超过 2 周不愈、发热、大片溃疡需就医；含激素口腔贴/膏不宜长期自行反复用

痔疮不适​

项目内容
常见药物fiber / 膳食纤维；stool softener / 软便剂；hydrocortisone/lidocaine / 氢化可的松/利多卡因外用
常见商品名Preparation H、马应龙、太宁
主要用途软便、减轻局部炎症/疼痛/瘙痒
注意便血先排除其他原因；长期/大量出血就医

常见处方药家族​

这些药通常不应自行购买或调整剂量，但理解名称有助于看处方。

高血压​

项目内容
常见类别CCB / 钙通道阻滞剂；ACEI；ARB
常见药物amlodipine / 氨氯地平；nifedipine / 硝苯地平；enalapril / 依那普利；losartan / 氯沙坦；valsartan / 缬沙坦
常见商品名Norvasc / 络活喜、拜新同、科素亚、代文
注意ACEI 可干咳；孕期禁用部分降压药；关注钾和肾功能

糖尿病​

项目内容
常见类别biguanide / 双胍类；SGLT2 inhibitor；GLP-1 RA
常见药物metformin / 二甲双胍；empagliflozin / 恩格列净；dapagliflozin / 达格列净；semaglutide / 司美格鲁肽；liraglutide / 利拉鲁肽
常见商品名Glucophage / 格华止、Jardiance / 欧唐静、Forxiga / 安达唐、Ozempic / 诺和泰、Wegovy、Saxenda / 善纤达
注意肾功能、脱水风险、胃肠反应、低血糖风险需结合具体方案评估

高 LDL-C​

项目内容
常见类别statin / 他汀
常见药物atorvastatin / 阿托伐他汀；rosuvastatin / 瑞舒伐他汀；simvastatin / 辛伐他汀
常见商品名Lipitor / 立普妥、Crestor / 可定、舒降之
注意关注肌痛、肝酶；与部分抗生素、抗真菌药、葡萄柚等可能有相互作用

痛风、高尿酸​

项目内容
急性发作常见药物colchicine / 秋水仙碱；NSAIDs 如 ibuprofen / 布洛芬、naproxen / 萘普生
长期降尿酸药物allopurinol / 别嘌醇；febuxostat / 非布司他
常见商品名秋水仙碱片、芬必得、Aleve、Zyloric、Uloric、非布司他片
注意急性期和长期管理策略不同；肾功能异常/药物相互作用需谨慎

哮喘、COPD​

项目内容
常见类别inhaled bronchodilator / 吸入支气管扩张剂；inhaled corticosteroid / 吸入糖皮质激素
常见药物salbutamol/albuterol / 沙丁胺醇；budesonide / 布地奈德；fluticasone / 氟替卡松
常见商品名Ventolin / 万托林、Pulmicort / 普米克、Flixotide / 辅舒酮
注意吸入方法很关键；急救药和长期控制药不要混淆

感染、抗生素​

项目内容
常见类别antibiotics / 抗生素
常见药物amoxicillin / 阿莫西林；azithromycin / 阿奇霉素；cephalexin / 头孢氨苄
常见商品名阿莫仙、希舒美、头孢氨苄胶囊
注意只对细菌感染；不要自行用、不要剩药复用、不要随意停药

风湿免疫​

项目内容
常见类别DMARDs / 改善病情抗风湿药
常见药物methotrexate / 甲氨蝶呤；hydroxychloroquine / 羟氯喹；leflunomide / 来氟米特
常见商品名Methotrexate、Plaquenil / 纷乐、爱若华
注意需监测血常规、肝肾功能等；见 DMARDs

感冒药为什么容易吃重复？​

很多复方感冒药同时包含退烧止痛、抗组胺、镇咳、减充血成分。再额外吃 acetaminophen / 对乙酰氨基酚、ibuprofen / 布洛芬或其他感冒药，可能造成重复用药或超量。

抗生素能治感冒、咽痛、腹泻吗？​

  • 抗生素只针对细菌感染，对普通病毒感冒无效。
  • 咽痛、腹泻也不等于细菌感染；是否需要抗生素要看病史、体征和检查。
  • 自行使用抗生素会增加过敏、副作用、耐药风险。

OTC 药可以一直吃吗？​

不建议。OTC 适合短期处理轻微症状。若需要长期或反复使用，说明可能有未解决病因，应咨询医生或药师。

drug vs medicine​

  • medicine
    • 指用于治疗、预防或改善健康状况的药物，语义更偏“医疗用途”。
  • drug
    • 泛指药物/化学活性物质，可指合法药物，也可指滥用药物。
  • all medicines are drugs, whereas not all drugs are medicines.
  • https://pharmafactz.com/what-is-the-difference-between-a-drug-and-a-medicine
