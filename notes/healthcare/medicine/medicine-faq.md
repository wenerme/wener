# Medical FAQ

:::warning

药物信息只用于理解常见药物类别，不构成诊断或处方建议。用药前应阅读说明书/Drug Facts label；孕期、哺乳期、儿童、老人、肝肾功能异常、慢病、多药合用、过敏史明确者，先问医生或药师。

:::

## 用药安全原则

- 看 **active ingredient / 通用名 / 成分名**，不要只看商品名；复方感冒药很容易和退烧止痛药重复成分。
- 商品名有地区差异，同一成分可能有多个品牌；优先核对中文通用名和英文通用名。
- 不要超说明书剂量、频次、天数；症状持续或加重应就医。
- 抗生素、激素、降压药、降糖药、抗凝药、精神科药物不要自行开始/停药。
- 儿童慎用成人药；2 岁以下不要自行用咳嗽/感冒药；儿童/青少年病毒感染时不要用 aspirin。
- 有高血压、心脏病、青光眼、前列腺增生者，慎用鼻减充血药。
- NSAIDs 如 ibuprofen / 布洛芬、naproxen / 萘普生可能刺激胃、影响肾功能、增加出血风险；胃溃疡、肾病、抗凝药使用者需谨慎。
- Acetaminophen / paracetamol / 对乙酰氨基酚/扑热息痛过量可伤肝；感冒药、止痛药、退烧药里可能重复包含。

## 发热、头痛、轻中度疼痛

| 项目       | 内容                                                     |
| ---------- | -------------------------------------------------------- |
| 常见药物   | acetaminophen/paracetamol / 对乙酰氨基酚/扑热息痛        |
| 常见商品名 | Tylenol / 泰诺、Panadol / 必理通、泰诺林                 |
| 主要用途   | 退热、头痛、轻中度疼痛                                   |
| 注意       | 注意每日总量；肝病、长期饮酒、复方感冒药同用时更容易超量 |

- FDA 标注成人和 12 岁以上 24 小时总量不应超过 4000 mg；不同商品说明书可能更保守，优先按说明书。
- 儿童剂量按体重和说明书/医生建议，不要用成人剂量估算。
- 如果高热持续、精神差、呼吸困难、皮疹、颈部强直，应及时就医。

## 疼痛伴炎症、牙痛、痛经、关节痛

| ---------- | --------------------------------------------------------------- |
| 常见药物   | ibuprofen / 布洛芬；naproxen / 萘普生                           |
| 常见商品名 | Advil、Motrin、芬必得；Aleve                                    |
| 药物类别   | NSAIDs / 非甾体抗炎药                                           |
| 注意       | 胃病、肾病、抗凝药、孕 20 周后/孕晚期慎用；不要多种 NSAIDs 叠加 |

- FDA 建议孕 20 周或之后避免 NSAIDs，除非医生明确建议；30 周后通常应避免。
- 疼痛伴红肿热痛、外伤变形、牙龈化脓、痛经突然加重，应找原因而不是只止痛。

## 感冒药为什么容易吃重复？

很多复方感冒药同时包含退烧止痛、抗组胺、镇咳、减充血成分。再额外吃 acetaminophen / 对乙酰氨基酚、ibuprofen / 布洛芬或其他感冒药，可能造成重复用药或超量。

## 抗生素能治感冒、咽痛、腹泻吗？

- 抗生素只针对细菌感染，对普通病毒感冒无效。
- 咽痛、腹泻也不等于细菌感染；是否需要抗生素要看病史、体征和检查。
- 自行使用抗生素会增加过敏、副作用、耐药风险。

## OTC 药可以一直吃吗？

不建议。OTC 适合短期处理轻微症状。若需要长期或反复使用，说明可能有未解决病因，应咨询医生或药师。

## drug vs medicine

- medicine
  - 指用于治疗、预防或改善健康状况的药物，语义更偏“医疗用途”。
- drug
  - 泛指药物/化学活性物质，可指合法药物，也可指滥用药物。
- all medicines are drugs, whereas not all drugs are medicines.
