# 咽炎/咽喉炎

- 英文：pharyngitis / sore throat
- 类型：咽部炎症或刺激症状，常见原因包括病毒感染、细菌感染、过敏后鼻滴流、胃食管反流、烟酒粉尘刺激、用嗓过度。
- 相关
  - [医学 FAQ：咽喉痛、咽喉发炎](../medicine/medicine-faq.md#咽喉痛咽喉发炎)

## TL;DR

- 多数急性咽痛为病毒或刺激所致，不等于细菌感染。
- 处理重点是补水、休息、减少刺激、止痛退热和处理诱因。
- 慢性反复要关注鼻炎后鼻滴流、胃食管反流、烟酒粉尘和用嗓习惯。
- 呼吸困难、吞咽困难、高热脓点、持续声音嘶哑等需要就医。

## 症状

- 咽痛、吞咽痛、咽干、异物感。
- 咳嗽、声音嘶哑、清嗓。
- 感染时可伴发热、流涕、咳嗽、扁桃体肿大或脓点。
- 慢性反复时可能与鼻炎、反流、吸烟/粉尘、职业用嗓相关。

## 常见原因/诱因

- 病毒感染最常见。
- 细菌感染如 strep throat 需要医生判断。
- 过敏性鼻炎、鼻窦炎导致的 postnasal drip。
- 胃食管反流、烟酒、粉尘、干燥空气、用嗓过度。

## 检查

- 急性咽痛通常结合症状、体温、咽部表现判断。
- 怀疑链球菌感染时，可做快速抗原检测或咽拭子培养。
- 慢性反复或声音嘶哑时，耳鼻喉科可评估鼻腔、咽喉和声带。

## 治疗方向

### 非药物

- 休息、补水、避免烟酒和刺激性食物。
- 温盐水漱口、含片、保持空气湿润。
- 慢性反复时处理过敏、后鼻滴流、反流和用嗓习惯。

### 常见药物类别

| 项目       | 内容                                                             |
| ---------- | ---------------------------------------------------------------- |
| 常见药物   | 对乙酰氨基酚/布洛芬；lozenges / 含片；saline gargle / 温盐水漱口 |
| 常见商品名 | 西瓜霜含片、金嗓子、Strepsils                                    |
| 注意       | 咽痛不等于细菌感染；不要自行长期或反复使用抗生素                 |

## 什么时候就医

- 呼吸困难、吞咽困难、流口水、张口困难。
- 高热、扁桃体脓点、皮疹、颈部明显肿痛。
- 症状持续不缓解或反复发作。
- 持续声音嘶哑、咯血、体重下降、单侧疼痛或颈部包块。

## 预防/复发管理

- 避免烟酒、粉尘和过度用嗓。
- 过敏性鼻炎或胃食管反流控制不好时，优先处理基础诱因。
- 感染季节注意手卫生和休息。
