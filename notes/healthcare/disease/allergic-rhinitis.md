# 过敏性鼻炎

- 英文：allergic rhinitis / hay fever
- 类型：IgE 介导的鼻黏膜过敏性炎症，可为季节性或常年性。
- 相关
  - [医学 FAQ：过敏性鼻炎、荨麻疹、皮肤痒](../medicine/medicine-faq.md#过敏性鼻炎荨麻疹皮肤痒)

## TL;DR

- 典型表现是连续打喷嚏、清水样流涕、鼻痒、鼻塞。
- 常见诱因包括花粉、屋尘螨、宠物皮屑、霉菌和空气刺激物。
- 环境控制、鼻腔冲洗和规范鼻喷药物比反复临时通鼻更重要。
- 鼻减充血喷剂不宜连续长期使用。

## 症状

- 连续打喷嚏、清水样流涕、鼻痒、鼻塞。
- 可伴眼痒流泪、咽痒、咳嗽、后鼻滴流。
- 鼻塞明显时可影响睡眠、嗅觉和白天注意力。
- 儿童可表现为长期张口呼吸、揉鼻、打鼾。

## 常见原因/诱因

- 花粉、屋尘螨、宠物皮屑、霉菌。
- 冷空气、烟雾、空气污染、香水等刺激物可加重症状。
- 可与哮喘、湿疹、结膜炎等过敏性疾病共存。

## 检查

- 多数根据典型症状、季节性和诱因判断。
- 反复或需要明确过敏原时，可做皮肤点刺试验或血清特异性 IgE。
- 单侧鼻塞、反复鼻出血或脓涕面痛时，应排除其他鼻腔/鼻窦问题。

## 治疗方向

### 非药物

- 明确并减少暴露：清洁寝具、控制尘螨、减少花粉暴露、避免烟雾刺激。
- 生理盐水鼻喷/冲洗可作为辅助。
- 花粉季外出后清洗面部、鼻腔和头发，减少室内带入。

### 常见药物类别

| 项目       | 内容                                                                    |
| ---------- | ----------------------------------------------------------------------- |
| 常见药物   | loratadine / 氯雷他定；cetirizine / 西替利嗪；fexofenadine / 非索非那定；鼻喷糖皮质激素 |
| 常见商品名 | Claritin / 开瑞坦、Zyrtec / 仙特明、Allegra                             |
| 注意       | 第二代抗组胺药相对少嗜睡；鼻减充血喷剂不宜连续长期使用                 |

## 什么时候就医

- 鼻塞严重影响睡眠、学习或工作。
- 反复鼻窦炎、中耳炎、哮喘/喘息。
- 单侧鼻塞、反复鼻出血、脓涕伴面痛发热、嗅觉明显下降。
- 儿童长期张口呼吸、打鼾、注意力下降。

## 预防/复发管理

- 根据过敏原做环境控制，而不是只在发作时临时吃药。
- 花粉季提前管理症状。
- 合并哮喘、湿疹时需整体管理过敏体质。
