# 疾病

| abbr. | stand for            | meaning        |
| ----- | -------------------- | -------------- |
| RA    | rheumatoid arthritis | 类风湿性关节炎 |

| en               | cn     |
| ---------------- | ------ |
| tinnitus         | 耳鸣   |
| house dust mites | 屋尘螨 |

- [类风湿性关节炎](./rheumatoid-arthritis.md) / RA / rheumatoid arthritis
  - 风湿免疫；慢性自身免疫性关节炎、DMARDs、关节保护。
- [肩周炎](./adhesive-capsulitis.md) / adhesive capsulitis / frozen shoulder
  - 骨科/康复；疼痛期、冻结期、解冻期，活动度恢复。
- [汗疱疹](./dyshidrosis.md) / dyshidrosis
  - 皮肤；手足小水疱、瘙痒、反复发作。
- [脚气/足癣](./tinea-pedis.md) / tinea pedis / athlete's foot
  - 皮肤/真菌感染；足部浅表真菌感染、趾缝脱皮瘙痒、保持干燥和抗真菌治疗。
- [湿疹/皮炎](./eczema.md) / eczema / dermatitis
  - 皮肤/过敏；皮肤屏障受损、瘙痒红斑、保湿和避免刺激。
- [咽炎/咽喉炎](./pharyngitis.md) / pharyngitis / sore throat
  - 耳鼻喉/感染；咽痛咽干、病毒/刺激常见、警惕高热脓点和呼吸吞咽困难。
- [过敏性鼻炎](./allergic-rhinitis.md) / allergic rhinitis / hay fever
  - 耳鼻喉/过敏；喷嚏流涕鼻痒鼻塞、过敏原控制和鼻喷药物管理。
- [胃食管反流/烧心](./gerd.md) / GERD / gastroesophageal reflux disease / heartburn
  - 消化；烧心反酸、饭后或平躺加重、生活方式调整和短期抑酸治疗。
