# 湿疹/皮炎

- 英文：eczema / dermatitis
- 类型：皮肤炎症反应，常与皮肤屏障受损、刺激物、过敏、干燥、出汗、压力相关。
- 相关
  - [医学 FAQ：湿疹、皮炎轻度瘙痒](../medicine/medicine-faq.md#湿疹皮炎轻度瘙痒)

## TL;DR

- 湿疹/皮炎不是单一疾病，而是一组皮肤炎症表现。
- 典型表现是瘙痒、红斑、干燥、脱屑，也可渗液、结痂。
- 保湿修复屏障和避免诱因是基础。
- 外用激素要按部位、强度、天数短期使用；面部、眼周、儿童、孕期需谨慎。

## 症状

- 瘙痒、红斑、干燥、脱屑、丘疹或小水疱。
- 急性可渗液、结痂。
- 慢性可皮肤增厚、粗糙、色素改变。
- 抓破后可能继发细菌感染。

## 常见原因/诱因

- 皮肤屏障受损、干燥、频繁清洁或热水烫洗。
- 香精、清洁剂、消毒剂、染发剂、金属、橡胶等刺激或接触过敏。
- 出汗、压力、季节变化、过敏体质。
- 部分部位反复时需排除真菌感染、银屑病等。

## 检查

- 多数根据皮疹形态、部位和诱因判断。
- 怀疑接触性皮炎时，可考虑斑贴试验。
- 不典型、反复或治疗无效时，应由皮肤科评估是否需要真菌检查、皮肤镜或其他检查。

## 治疗方向

### 非药物

- 保湿修复屏障，减少热水、香精、刺激性清洁剂和频繁消毒。
- 识别并避开诱因，例如金属、染发剂、橡胶、清洁剂、护肤品。
- 避免抓挠，必要时修短指甲。

### 常见药物类别

| 项目       | 内容                                                                                      |
| ---------- | ----------------------------------------------------------------------------------------- |
| 常见药物   | emollient / 润肤剂；hydrocortisone / 氢化可的松；antihistamine / 抗组胺药                 |
| 常见商品名 | Vaseline / 凡士林、Cortaid、氢化可的松乳膏；尤卓尔为丁酸氢化可的松，按说明书/医嘱短期使用 |
| 注意       | 面部/眼周/儿童/感染破溃慎用激素；大片扩散或渗液需就医                                     |

## 什么时候就医

- 大片扩散、明显渗液化脓、疼痛发热。
- 反复影响睡眠和生活。
- 面部、眼周、外阴、婴幼儿、孕期、免疫低下。
- 外用药无效或停药很快复发。

## 预防/复发管理

- 日常保湿，洗澡后及时涂润肤剂。
- 减少刺激性清洁剂和香精类产品。
- 记录发作前接触物、食物、环境和压力变化。
