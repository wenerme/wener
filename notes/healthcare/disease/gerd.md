# 胃食管反流/烧心

- 英文：GERD / gastroesophageal reflux disease; heartburn
- 类型：胃内容物反流到食管导致的烧心、反酸或相关症状。
- 相关
  - [医学 FAQ：胃酸、烧心](../medicine/medicine-faq.md#胃酸烧心)

## TL;DR

- 典型表现是胸骨后烧灼感、反酸，饭后、弯腰、平躺时加重。
- 生活方式调整是基础：避免睡前进食、减少诱发食物、必要时减重。
- 短期可用抗酸药、海藻酸盐、H2 blocker 或 PPI，但长期用药应评估。
- 胸痛要先排除心血管急症。

## 症状

- 胸骨后烧灼感、反酸、口苦/酸味、嗳气。
- 常在饭后、弯腰、平躺时加重。
- 可伴慢性咳嗽、清嗓、咽喉异物感、声音嘶哑。
- 长期反流可能影响牙釉质或睡眠。

## 常见原因/诱因

- 过饱、夜宵、餐后立刻躺下。
- 肥胖、妊娠、腹压增加。
- 烟酒、咖啡、薄荷、巧克力、高脂/辛辣食物。
- 部分药物或食管裂孔疝等因素。

## 检查

- 典型轻症可先根据症状和治疗反应判断。
- 有警示症状或长期反复时，医生可能评估胃镜、幽门螺杆菌、食管 pH 监测等。
- 胸痛、胸闷等需先排除心血管问题。

## 治疗方向

### 非药物

- 避免睡前 2–3 小时进食。
- 少量多餐，减少诱发食物，必要时减重。
- 夜间症状明显可抬高床头。
- 避免餐后立刻躺下或弯腰用力。

### 常见药物类别

| 项目       | 内容                                                                                |
| ---------- | ----------------------------------------------------------------------------------- |
| 常见药物   | antacid / 抗酸药；alginate / 海藻酸盐；famotidine / 法莫替丁；omeprazole / 奥美拉唑 |
| 常见商品名 | Tums、Gaviscon、Pepcid、Prilosec、达喜、洛赛克                                      |
| 注意       | PPI 不建议无评估长期吃；胸痛需先排除心血管急症                                      |

## 什么时候就医

- 胸痛、胸闷、出汗、放射痛。
- 吞咽困难/疼痛、呕血、黑便、贫血、体重下降、持续呕吐。
- 症状频繁、夜间反流明显、长期依赖药物或停药复发。

## 预防/复发管理

- 记录并减少个人诱发食物和生活习惯。
- 控制体重，避免睡前进食。
- 反复咽喉症状、慢性咳嗽或声音嘶哑时，也应考虑反流因素。
