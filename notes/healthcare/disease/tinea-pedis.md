# 脚气/足癣

- 英文：tinea pedis / athlete's foot
- 类型：足部浅表真菌感染，常见于脚趾缝、足底、足侧。
- 相关
  - [医学 FAQ：脚气、足癣、真菌感染](../medicine/medicine-faq.md#脚气足癣真菌感染)

## TL;DR

- 常见于脚趾缝、足底、足侧。
- 典型表现是瘙痒、脱皮、裂口、水疱或足底鳞屑。
- 处理重点是抗真菌药加保持干燥，不是只抹一次药。
- 反复、无效、合并趾甲改变或糖尿病/免疫低下时应就医。

## 症状

- 脚趾缝潮湿、发白、脱皮、裂口、瘙痒或刺痛。
- 足底鳞屑、角化增厚，或出现小水疱。
- 可合并脚臭。
- 若趾甲变厚、变黄、碎裂，需考虑甲真菌病。

## 常见原因/诱因

- 皮肤癣菌等真菌感染。
- 足部潮湿、出汗多、鞋袜不透气。
- 公共浴室、泳池、更衣室赤脚接触。
- 家人或同住者存在足癣、甲真菌病。

## 检查

- 多数可根据表现初步判断。
- 反复或不典型时，皮肤科可做真菌镜检/培养。
- 需与湿疹、接触性皮炎、汗疱疹、银屑病等区分。

## 治疗方向

### 非药物

- 保持足部干燥，洗脚后擦干脚趾缝。
- 勤换袜，鞋子通风轮换。
- 不共用拖鞋、毛巾、指甲刀。
- 公共浴室、泳池、更衣室尽量穿拖鞋。

### 常见药物类别

| 项目       | 内容                                                                                    |
| ---------- | --------------------------------------------------------------------------------------- |
| 常见药物   | terbinafine / 特比萘芬；clotrimazole / 克霉唑；miconazole / 咪康唑；tolnaftate / 托萘酯 |
| 常见商品名 | Lamisil / 兰美抒、Lotrimin、Daktarin / 达克宁                                           |
| 注意       | 症状好转后通常仍需按说明继续用一段时间；避免过早停药复发                                |

## 什么时候就医

- 2–4 周规范处理仍无改善。
- 反复发作、范围扩大、明显红肿疼痛渗液。
- 糖尿病、免疫低下、足部伤口。
- 疑似甲真菌病或需要口服抗真菌药。

## 预防/复发管理

- 鞋袜保持干燥透气。
- 运动后及时清洁和更换袜子。
- 家庭成员同时有足癣/甲真菌病时，应一起处理传播源。
