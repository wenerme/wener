## RF 类风湿因子测定

0-14.0 KIU/L

原因

90% 的 RA 患者 RF 为阳性

## ESR 红细胞沉降率

0-20 mm/h

原因

1. 生理性 - 月经、妊娠， 12 以下儿童、60 以上老人
2. 病理性
   1. 炎症性疾病
   2. 组织损伤及坏死
   3. 恶性肿瘤
   4. 高血球蛋白症
   5. 部分贫血患者
   6. 高胆固醇血症、糖尿病
