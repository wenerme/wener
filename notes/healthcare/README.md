# 健康

- Healthcare - 医疗保健
  - 涵盖医学服务、护理、康复、预防保健、健康教育、公共卫生和健康管理。
- Medicine - 医学
  - 研究人体健康、疾病、诊断、治疗和预防的科学体系。
- 免责声明
  - 笔记用于建立索引和理解常识，不替代医生诊断、处方或急诊处理。

## 组织边界

- `disease/`：回答“这是什么病、如何判断、什么时候就医”。
- `medicine/`：回答“这个症状常见用什么药、如何安全用药”。
- `drug/`：回答“这类药是什么、机制/风险/监测是什么”。
- `metrics/`：回答“化验指标是什么、单位和组合怎么看”。

## 页面拆分原则

- 一个疾病/病症如果会反复查，应建 `disease/<name>.md`。
- 一个药物类别如果有监测、禁忌、相互作用，应建 `drug/<class>.md`。
- 一个检验指标如果解释较多，应放到 `metrics/` 或在 `metrics/README.md` 中保留速查。
- `FAQ` 页面只保留入口答案，详细内容链接到 disease/drug/metrics。
