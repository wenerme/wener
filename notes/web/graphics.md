# Web Graphics

## Charting

- [Chart.js](https://www.chartjs.org/)
  - 8 pre-built chart types: line, bar, pie, polar, bubble, scatter, area and mixed
- [gionkunz/chartist-js](https://github.com/gionkunz/chartist-js)
  - 10KB with zero dependencies
  - chart types: line, bar and pie
  - SVG
  - React and Angular
- [britecharts/britecharts](https://github.com/britecharts/britecharts)
  - D3
  - line, bar, donut, bullet, scatter plot, sparkline and step
- [TargetHolding/taucharts](https://github.com/TargetHolding/taucharts)
  - D3
  - line, bar, scatterplot, area and facet charts.
  - [The Grammar of Graphics](https://target.georiot.com/Proxy.ashx?tsid=8429&GR_URL=https%3A%2F%2Fwww.amazon.com%2FGrammar-Graphics-Statistics-Computing%2Fdp%2F0387245448%3Ftag%3Dhawk-future-20%26ascsubtag%3Dcbq-4296258895163873000-20) into 'Taucharts Language'

## Data Visualization & Drawing

- [d3/d3](https://github.com/d3/d3)
  - Data-Driven Documents
- [jonobr1/two.js](https://github.com/jonobr1/two.js)
  - two-dimensional drawing on the web
  - SVG, Canvas and WebGL
- [williamngan/pts](https://github.com/williamngan/pts)
  - space, form and point
  - A library for visualization and creative-coding
- [adobe-webplatform/Snap.svg](https://github.com/adobe-webplatform/Snap.svg)
  - [Snap.svg](http://snapsvg.io/)
  - working with your SVG assets as easy as jQuery makes working with the DOM

## 3D & WebGL

- [pixijs/pixijs](https://github.com/pixijs/pixijs)
  - 2D WebGL renderer
- [metafizzy/zdog](https://github.com/metafizzy/zdog)
  - [Zdog](https://zzz.dog/)
  - pseudo-3D experiences
  - Canvas or SVG
