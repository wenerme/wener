# Web Charting

- [leeoniya/uPlot](https://github.com/leeoniya/uPlot)
  - A small, fast chart for time series, lines, areas, ohlc & bars

## Considerations

- **Data**: Static vs. Dynamic
- **Interaction**: High interactivity required?
- **Mobile**: Support needed?
- **Volume**: Dataset size
- **Lib Type**: Standalone vs. Framework-based (e.g., React)
- **Customization**
- **Chart Types**
- **Implementation**:
  - SVG
  - Canvas
  - WebGL
- **License**:
  - Most JS libs are MIT
  - Some commercial ones are GPL
