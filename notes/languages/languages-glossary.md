## Natural Language

### 语音学 (Phonetics/Phonology)

| en                 | cn              | 说明                                                  |
| ------------------ | --------------- | ----------------------------------------------------- |
| Initial/Onset      | 声母            | 音节开头的辅音部分，如 "b" in "ba"                    |
| Medial             | 介音            | 韵母中位于主要元音前的过渡音，如 i、u、ü             |
| Nucleus            | 主要元音        | 韵母的核心元音部分，如 a、o、e、i、u、ü              |
| Coda               | 韵尾            | 韵母末尾的辅音部分，如 n、ng                         |
| Final/Rhyme        | 韵母            | 音节中声母后的部分，包含介音+主要元音+韵尾            |
| Consonant          | 辅音/子音       | 发音时气流受阻的音，如 b、p、m、f                     |
| Vowel              | 元音/母音       | 发音时气流不受阻的音，如 a、o、e、i、u、ü             |
| Syllable           | 音节            | 语音的最小单位，由声母+韵母组成                       |
| Integral Syllable  | 整体认读音节    | 声韵不能分开拼读的特殊音节，如 zh、ch、sh、r、z、c、s |
| Tone               | 声调            | 音节的高低升降变化，如普通话的四声                    |
| Consonant Cluster  | 复辅音/辅音丛   | 连续的多个辅音，如 "str" in "street"                  |
| Nasal              | 鼻音            | 通过鼻腔发出的音，如 m、n、ng                         |
| Lateral            | 边音            | 舌头侧边发音，如 l                                    |
| Plosive/Stop       | 塞音            | 完全阻塞气流后突然释放，如 p、t、k                    |
| Fricative          | 擦音            | 气流通过窄缝摩擦发出，如 f、s、sh                     |
| Affricate          | 塞擦音          | 塞音+擦音组合，如 zh、ch、j                           |
| Approximant        | 近音            | 发音器官接近但不完全阻塞，如 r、w、y                  |
| Diphthong          | 双元音/复合元音 | 两个元音的滑动，如 ai、ei、ao                         |
| Monophthong        | 单元音          | 单一的元音，如 a、o、e                                |
| Syllabic Consonant | 成音节辅音      | 能单独成音节的辅音，如 m、n、l、r                     |

### 音韵学 (Phonology)

| en            | cn         | 说明                     |
| ------------- | ---------- | ------------------------ |
| Phoneme       | 音位       | 最小的语音区别单位       |
| Allophone     | 音位变体   | 同一音位的不同发音变体   |
| Minimal Pair  | 最小对立体 | 只在一个音位上不同的词对 |
| Assimilation  | 同化       | 一个音受邻近音影响而改变 |
| Dissimilation | 异化       | 相邻音变得更不相似       |
| Elision       | 省略       | 音的脱落                 |
| Epenthesis    | 增音       | 插入额外音素             |
| Metathesis    | 音位互换   | 音的位置互换             |
