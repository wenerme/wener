# OmniVision OV2640

2 Megapixel Color CMOS UXGA (1632x1232) Image Sensor with OmniPixel2 Technology.

## Key Specs

- **Resolution**: 2 MP (1600 x 1200)
- **Formats**: YUV422, YCbCr422, RGB565, JPEG
- **Interface**: SCCB (I2C compatible) for control, DVP (8-bit) for data
- **Voltage**: Analog 2.5-3.0V, Digital 1.3V, I/O 1.7-3.3V

## Usage

Commonly used in ESP32-CAM modules.
