# 瑞芯微电子 SoC (Rockchip SoCs) {#rockchip-socs}

瑞芯微电子 (Rockchip) 是一家位于福州的中国无晶圆厂半导体公司，专注于为平板电脑、笔记本电脑、流媒体盒和嵌入式系统提供基于 ARM 的片上系统 (SoC)。

- [Rockchip 官网](http://www.rock-chips.com/)

## 流行 SoC 系列 {#popular-soc-series}

### 高性能与桌面 {#high-performance-desktop}

- **RK3588**: 旗舰 8 核 SoC (4x Cortex-A76 + 4x Cortex-A55)，配备 6 TOPS NPU。用于高端 SBC (Orange Pi 5, Rock 5B) 和边缘计算。
- **RK3399**: 六核 (2x Cortex-A72 + 4x Cortex-A53)。前代旗舰，流行于 Chromebook 和高性能 SBC (Pine64 RockPro64)。

### 中端与 IoT {#mid-range-iot}

- **RK3566 / RK3568**: 四核 Cortex-A55 带 NPU。广泛用于现代入门级 SBC、NAS 和 NVR (例如 Quartz64)。
- **RK3288**: 首批支持 4K H.264/H.265 硬件解码的流行 Rockchip SoC 之一。常见于许多旧款平板电脑和 Tinker Board。

### 入门与多媒体 {#budget-multimedia}

- **RK3128 / RK3229**: 四核 Cortex-A7 设计，用于低成本电视盒和特定工业终端 (例如华为 EC6108V9A)。

## 技术特性 {#technical-features}

- **GPU Integration**: 通常使用 ARM Mali GPU (例如 Mali-G610, Mali-T860)。
- **Video Processing**: 强大的硬件加速视频解码支持 (H.264, H.265, VP9)。
- **Open Source Support**: Rockchip 为其许多芯片提供了重要的文档和主线 Linux 内核支持，使其在开发者社区中很受欢迎。
