# 通信硬件术语表 (Telecommunications Hardware Glossary) {#telecom-hardware-glossary}

通信基础设施与硬件相关技术术语的定义与解释。

## SLIC (Subscriber Line Interface Card) {#slic}

用户线路接口卡 (SLIC) 是将电信网络 (PSTN) 与用户本地环路连接的集成电路或模块化电路。

- **Primary Function (主要功能)**: 它将电话的模拟语音信号转换为网络的数字数据，反之亦然。它还为传统模拟电话提供信令 (振铃、拨号音) 和馈电 (电池馈电)。
- **Form Factor (外形规格)**: SLIC 可以是独立 IC 或高密度板卡，在中心局 (CO) 或远程终端部署中服务多达 96 个用户。
- **Reference**: [SLIC Definition (Computer Hope)](https://www.computerhope.com/jargon/s/slic.htm)

## 相关术语 {#related-terms}

- **SLAC (Subscriber Line Audio Circuit / 用户线路音频电路)**: 常与 SLIC 配合使用。SLAC 处理数字信号处理 (DSP) 和编解码功能，而 SLIC 处理高压大电流模拟接口。
- **PSTN (Public Switched Telephone Network / 公共交换电话网)**: 世界各国、地区或本地电话运营商运营的电路交换电话网络的集合。
- **CPE (Customer Premises Equipment / 客户驻地设备)**: 位于用户驻地并连接到运营商电信信道的任何终端和相关设备 (如路由器、ONT、电话)。
- **Local Loop (本地环路)**: 从用户家到电信服务提供商中心局的物理线路连接。
