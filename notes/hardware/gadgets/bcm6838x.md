# Broadcom BCM6838x PON 处理器 {#broadcom-bcm6838x-pon-processors}

BCM6838x 系列是 Broadcom 设计的第四代无源光网络 (PON) 处理器系列，专为 GPON 和 EPON 光网络终端 (ONT) 设计。

## 概述 {#overview}

BCM6838x 建立在 Broadcom 的宽带硬件和软件架构之上。它以高集成度和高性价比著称，通常允许双层 PCB 设计。

- [Broadcom BCM6838x 产品页面](https://www.broadcom.com/products/broadband/xpon/bcm6838x-series)
- [数据手册 / 元件搜索](https://www.transistorthyristor.com/circuits?search=BCM683851FSBG)

## 主要特性 {#key-features}

- **High Integration (高集成度)**: 集成千兆以太网 (GbE)PHY、USB、SLAC (用户线路音频电路) 和 SIM 卡控制器。
- **Power Management (电源管理)**: 单电源供电运行。
- **WiFi Acceleration (WiFi 加速)**: 支持单频或并发双频 WiFi 硬件加速。
- **Unified Software (统一软件)**: 使用 Broadcom 的统一软件环境 (BcmSDK/BcmComm)，在 xDSL、EPON 和 GPON 产品之间共享。
- **Low BOM Cost (低 BOM 成本)**: 专为大批量 CPE (客户驻地设备) 的最低物料清单成本设计。
- **Market Presence (市场表现)**: 属于年部署量超过 5000 万台 CPE 的平台系列。

## 常见型号 {#common-models}

- **BCM68380**: 标准 ONT 的基础型号。
- **BCM61838**: 针对不同地区/标准的特殊变体。

## 硬件示例 {#hardware-examples}

- 常见于 ISP 提供的各种华为、中兴和诺基亚/阿尔卡特-朗讯 ONT/ONU 设备中。
- 通常搭配 Broadcom WiFi 芯片组 (如 BCM43xx)。
