# 华为 EC6108V9 系列 (悦盒) {#huawei-ec6108v9-series}

华为为中国电信运营商 (电信、联通、移动) 生产的流行 Android 机顶盒系列。

## 硬件规格 {#hardware-specifications}

### EC6108V9A

- **CPU**: Rockchip [RK3128](http://www.rock-chips.com/a/cn/product/rk31/2014/0924/524.html) (2x 1.2 GHz, 也有四核版本)。
- **RAM**: 1GB DDR3.
- **Storage**: 4GB eMMC.
- **OS**: Android 4.4 / 4.2.2.
- **Output**: 最高 1080p @ 60Hz.
- **Interfaces**:
  - 12V 1A DC 电源。
  - Micro SD 卡槽。
  - USB 2.0 x 2.
  - HDMI + AV 接口。
  - 10/100M 以太网 (LAN)。

### 其他变体 (基于海思) {#other-variants-hisilicon}

- **V93/V94/V97/V9C/V9U**: 通常基于海思 (HiSilicon) HI3798M 芯片组。

## 技巧与故障排除 {#tips-troubleshooting}

### 基本操作 {#basic-operations}

- **Recovery Mode (恢复模式)**:
  - 启动时通过电源开关快速开关。
  - 或者按住遥控器的左右方向键。
- **Update Shortcut (强制升级)**: 启动时按住 "Home" 键，触发从 USB 驱动器的 `upgrade/update.img` 进行强制更新。
- **Default IP**: `172.16.20.14`.
- **Browser Shortcut**: `Win + B` (如果使用物理键盘)。

### 常用密码/操作码 {#common-passcodes}

- **Settings/Unlock Codes (设置/解锁码)**:
  - `6321`
  - `10010`
  - `131022`
