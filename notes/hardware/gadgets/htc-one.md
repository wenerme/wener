# HTC One (M7 / M8) {#htc-one}

HTC One 系列以其全铝一体成型机身设计代表了高端 Android 智能手机设计的重要里程碑。

## HTC One M7 硬件规格 (经典版) {#htc-one-m7-hardware-specs}

- **Model Variant**: HTC 6500L (Verizon 版).
- **CPU**: Qualcomm [Snapdragon 600](https://www.gsmarena.com/htc_one-5313.php) (四核 1.7 GHz Krait 300).
- **GPU**: Adreno 320.
- **RAM**: 2GB DDR2.
- **Storage**: 32GB/64GB (不可扩展).
- **Screen**: 4.7" Super LCD3, 1080 x 1920 像素 (~469 ppi).

## HTC One M8 硬件规格 (继任者) {#htc-one-m8-hardware-specs}

- **CPU**: Qualcomm Snapdragon 801 (四核 2.3 GHz).
- **RAM**: 2GB DDR3.
- **Storage**: 16GB/32GB + 支持 MicroSD.
- **Screen**: 5.0" Super LCD3, 1080 x 1920 像素.

## 关键技术 {#key-technologies}

- **UltraPixel Camera**: 更大的 2.0 µm 像素尺寸，用于更好的低光性能 (4MP 分辨率)。
- **BoomSound**: 双前置立体声扬声器。
- **Sense UI**: HTC 定制的 Android 皮肤。
