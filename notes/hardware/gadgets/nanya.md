# 南亚科技 (Nanya Technology) {#nanya-technology}

南亚科技 (Nanya Technology Corporation) 是一家领先的台湾半导体制造商，专注于动态随机存取存储器 (DRAM) 的设计和制造。

- [Nanya 官网](http://www.nanya.com/)

## NT5CB64M16FP 技术规格 {#nt5cb64m16fp-technical-specifications}

NT5CB64M16FP 是一款 1 G-bit (128MB) DDR3 SDRAM 芯片，通常见于路由器、机顶盒和笔记本电脑等嵌入式设备中。

### 主要特性 {#key-features}

- **Capacity**: 1 G-bit (64M x 16 bits).
- **Type**: DDR3 SDRAM.
- **Clock Speed**: 800 MHz (DDR3-1600).
- **Latency**: 10-10-10.
- **Voltage**: 1.5V (标准 DDR3).
- **Package**: 78-ball TFBGA.
- [NT5CB64M16FP 产品页面 (Nanya)](http://www.nanya.com/en/Product/3756/NT5CB64M16FP-DII)

### 组织结构 {#organization}

- **Depth/Width**: 64M x 16 bits.
- **Clock**: 同步 DRAM 接口。

## 南亚部件编号系统 (解析) {#nanya-part-numbering-system}

示例: `NT 5C B 64M16 F P - DH`

| Segment   | Meaning             | Details                                       |
| :-------- | :------------------ | :-------------------------------------------- |
| **NT**    | Manufacturer        | Nanya Technology                              |
| **5C**    | Product Family      | DDR3 DRAM                                     |
| **B**     | Interface & Voltage | `B` = SSTL_15 (1.5V) / `C` = SSTL_135 (1.35V) |
| **64M16** | Organization        | Depth (64M) \* Width (16 bits) = 1Gb total    |
| **F**     | Revision            | `F` = 6th Generation (Revision F)             |
| **P**     | Package             | `P` = 78 Ball TFBGA                           |
| **-DH**   | Speed Grade         | `DH` = DDR3-1600 (10-10-10 latency)           |

## 相关产品 {#related-products}

南亚提供广泛的存储解决方案：

- **Computing**: 用于 PC 和服务器的 DDR3, DDR4, 和 DDR5。
- **Consumer**: 用于网络、机顶盒和数字电视的高速 DRAM。
- **Mobile/Low Power**: 用于便携设备的 LPDDR3, LPDDR4。
- **Industrial/Automotive**: 专用的高耐用性存储模块。
