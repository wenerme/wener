# HTC One E8 (M8Sw) {#htc-one-e8}

HTC One E8 是 2014 年发布的 HTC One M8 旗舰产品的廉价塑料版本。

## 硬件规格 {#hardware-specifications}

- **Model Variant**: M8Sw (中国联通版).
- **CPU**: Qualcomm [Snapdragon 801](<https://www.gsmarena.com/htc_one_(e8)-6397.php>) (四核 2.5 GHz Krait 400).
- **GPU**: Adreno 330.
- **RAM**: 2GB DDR3.
- **Storage**: 16GB eMMC, 可通过 MicroSD 扩展 (最大 128GB).
- **Screen**: 5.0" Super LCD3, 1080 x 1920 像素 (~441 ppi).
- **OS**: 最初搭载 Android 4.4.2 (KitKat) 和 Sense 6.

## 特性 {#features}

- 聚碳酸酯一体成型设计。
- 双前置 BoomSound 扬声器。
- 13MP 主摄像头 (取代了 M8 的 UltraPixel 摄像头)。
- 5MP 前置摄像头。
