# Netgear R6300 v2 Router

The Netgear R6300 v2 is a popular high-end AC1750 router based on the Broadcom BCM4708 SoC. It is well-supported by the open-source community, making it a favorite for custom firmware.

## Hardware Specifications

- **SoC**: Broadcom BCM4708A0 (800MHz, Dual-Core).
- **RAM**: 256MB DDR3.
- **Flash**: 128MB NAND.
- **Wireless**:
  - 2.4GHz: BCM4331 (up to 450Mbps).
  - 5GHz: BCM4360 (up to 1300Mbps).
- **Ports**: 1 x WAN, 4 x LAN (Gigabit), 1 x USB 2.0, 1 x USB 3.0.

## Alternative Firmwares

### OpenWrt

Full Linux-based customization. Note that Broadcom wireless drivers are often proprietary, so check for current support status.

- [OpenWrt Table of Hardware: R6300 v2](https://openwrt.org/toh/netgear/netgear_r6300_v2)
- [OpenWrt Hardware Data](https://openwrt.org/toh/hwdata/netgear/netgear_r6300_v2)

### AdvancedTomato (Tomato by Shibby fork)

Known for its beautiful, modern web UI and ease of use compared to stock firmware or OpenWrt.

- [AdvancedTomato Download for R6300 v2](https://advancedtomato.com/downloads/router/r6300v2)
- - [Jackysi/advancedtomato](https://github.com/Jackysi/advancedtomato)
    - C, JavaScript (Bootstrap, jQuery), GPL
    - A fork of the Tomato firmware with a modern, flat-design GUI. Provides advanced Monitoring, QoS, and Wireless controls.

## Features

- **Beamforming+**: Improved range and performance for 5GHz devices.
- **ReadySHARE**: Easy USB storage and printer sharing.
- **VLAN Support**: Available via custom firmware for advanced home networking.
