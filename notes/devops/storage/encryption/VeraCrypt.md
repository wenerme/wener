# VeraCrypt

[VeraCrypt](https://github.com/veracrypt/VeraCrypt) is an open-source utility used for on-the-fly encryption (OTFE). It is based on TrueCrypt 7.1a.

## Information

- [Wikipedia - VeraCrypt](https://en.wikipedia.org/wiki/VeraCrypt)
- [Alpine Linux Package](https://pkgs.alpinelinux.org/package/edge/main/x86/truecrypt)

## Usage

### macOS Command Line

For more details, see the [Command Line Usage](https://www.veracrypt.fr/en/Command%20Line%20Usage.html) documentation.

```bash
/Applications/VeraCrypt.app/Contents/MacOS/VeraCrypt --help
```

### Notes

- 基于 7.1a
- 支持 LUKS (via cryptsetup on Linux)
